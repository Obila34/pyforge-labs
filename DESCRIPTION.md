# PyForge Labs

> From zero to model — hands-on Python codelabs for every level.

PyForge Labs is a curated repository of step-by-step coding sessions designed for learners at all stages — whether you're writing your first `print("Hello, World!")` or building and deploying machine learning pipelines.

---
---

##  Getting Started

1. Clone the repo
```bash
   git clone https://github.com/your-username/pyforge-labs.git
   cd pyforge-labs
```

2. Set up a virtual environment
```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

3. Install dependencies for a session
```bash
   pip install -r <session-folder>/requirements.txt
```

4. Open the codelab notebook or script and follow along!
